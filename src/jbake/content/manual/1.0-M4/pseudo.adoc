CSS Pseudo-classes such as :hover, :active, :nth-child, etc.

[source,groovy]
input % hover { color blue }
li % nthChild('3n') { color blue }

Produces:

[source,groovy]
input:hover { color: Blue; }
li:nth-child(3n) { color: Blue; }

There are also shorthands like 'odd' and 'even':

[source,groovy]
li % odd { color blue }

Produces:

[source,groovy]
li:nth-child(odd) { color: Blue; }
